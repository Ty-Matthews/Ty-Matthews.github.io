To download these tutorial and reference pages, click on the link below. To run them locally with Netscape, go to File...Open, then change to the directory you've unzipped the files to. All of the links will work as long as the files are all kept in the same directory (with the exception of the maps I have stored on my web site.) If you get error messages such as "Netscape was unable to find www.yourserver.com. It does not have a DNS entry.", this is normal. It means your home site is pointing to an Internet site, which of course you can't reach unless you're online.
I've changed my own home page to the following, which then keeps Netscape from looking out to the Net for the default page:

file:///c:/duke/dukefaq.htm


Alternatively, you can change the General Preferences under the Options menu item so that Netscape begins with a blank page. The main page is dukefaq.htm, with duketext.htm being the text-only version. Many of my pages have been included on a Duke CD from Walnut Creek.
